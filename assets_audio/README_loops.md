# Audio Loops (TONOSAI)

Ovaj folder sadrži muzičke loop-ove korišćene u projektima TONOSAI.

## 🧬 Poreklo zvuka

- Kreirani ručno u Studio One 5/7
- Delimično generisani pomoću AI (TONOSAI improvizator)

## 🔊 Tehničke informacije

- Format: .ogg / .wav
- Tempo: 70–120 BPM
- Skale: C-dur, Lydian, pentatonska
